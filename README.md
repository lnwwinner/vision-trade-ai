# Vision Trade AI

AI-powered automated trading system using Vision AI, technical analysis, and reinforcement learning.

## Features
- Vision AI (Gemini / OpenCV)
- Technical Indicators (RSI, MACD, EMA)
- Decision Engine (Hybrid AI + Rule-based)
- Auto Execution (ADB Control)
- Self-learning loop (DQN / LSTM ready)

## Architecture
Screen Capture → Vision AI → Decision Engine → Execution → Learning Loop
